Proyecto realizado en compiladores
C++

Con el archivo er.txt se obtiene un AFN y se genera una tabla para el AFD.
La tabla se usa en el analisis sintáctico del lenguaje. 
Los archivos Digitos.txt, PalabrasReservadas.txt, Producciones.txt, progFuente.cpp y
 Reglas_semanticas.txt se usan
para la ejecucion del compilador-traductor.

Al ejecutar se generan 10 archivos:
Primeros.txt
Siguientes.txt
Tabla de errores.txt
Tabla de simbolos.txt
tabla de tansiciones AFD.txt
tabla de trasiciones AFN.txt
Tabla_de_Analisis_Sintactico.txt
Tira de tokens.txt
Coleccion_canonica.txt
Analisis_sintactico.txt
